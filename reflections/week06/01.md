## Questions

1. What is Component based architecture?
where you build something that can be used/accessed in other parts of your program
2. What are some benefits of Component based architecture?
you can scale web building fast no need to make things multiple times and keeps things simple
3. What are some drawbacks to Component based architecture?
it can get confusing when you need to pass information before they get injected when running a page that has components that need specific information to run
